# minicut

minicut is a practice browser video editor scaffold built with Vite, React, and TypeScript.

## Getting started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build a production bundle:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Current scaffold

The first editor shell includes:

- A media preview area.
- An import media button placeholder.
- A simple timeline placeholder with video and audio tracks.
- A right-side controls panel placeholder for future clip settings.
